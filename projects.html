Venn Diagram Algorithm

This is the C++ code that I used for the paper here. The Basics of what it does is it takes in a file of Venn diagrams and finds what it called the dual graph of the diagram which is then passed into my algorithm. My algorithm tries to find a Hamiltonian cycle in the graph and returns if it was successful. The code comes in a zip file that has all the code, a ReadMe file with running instructions, and a Bash file for the actual running of the program.

A Fast Way to Extend Simple Venn Diagrams

This is a paper that I wrote entirely in LaTeX for a final project in a topics course in algorithms. It covers a conjecture proposed by Peter Winkler and presents an algorithm that finds if a simple Venn diagram is extendable or not. The result was an algorithm that could extend the 3,430,404 6-Venn diagrams 90% of the time in one hour. The algorithm developed by the professor did an exhaustive search that took three days. Using my algorithm to eliminate 90% of the diagrams and then run the professors algorithm would reduce the total computation time drastically to do an exhaustive search. The code can be found in the code section under Venn Diagram Algorithm.
