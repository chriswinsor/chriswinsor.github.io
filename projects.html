A Fast Way to Extend Simple Venn Diagrams

This is a paper that I wrote entirely in LaTeX for a final project in a topics course in algorithms. It covers a conjecture proposed by Peter Winkler and presents an algorithm that finds if a simple Venn diagram is extendable or not. The result was an algorithm that could extend the 3,430,404 6-Venn diagrams 90% of the time in one hour. The algorithm developed by the professor did an exaustive search that took three days. Using my algorithm to eliminate 90% of the diagrams and then run the professors algorithm would reduce the total computation time drasticly to do an exhaustive search.
